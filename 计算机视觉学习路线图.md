## 📅 推荐学习计划

### 第 1-2 周：PyTorch + CNN 基础
```
Day 1-3:  PyTorch 基础 (Tensor, autograd, nn.Module)
Day 4-5:  用 PyTorch 实现 CNN 图像分类 (CIFAR-10)
Day 6-7:  迁移学习 (ResNet 微调)
Day 8-10: 看 CS231n Lecture 5-9 (CNN 原理)
Day 11-14: 动手学深度学习 CNN 章节
```

### 第 3-4 周：目标检测 (YOLO)
```
Day 1-3:  安装 Ultralytics，跑通 COCO 预训练
Day 4-7:  自己标注数据集，训练自定义 YOLOv8
Day 8-10: 学习检测原理 (Anchor, NMS, mAP)
Day 11-14: 尝试 RT-DETR，对比不同检测器
```

### 第 5-6 周：分割 + 跟踪
```
Day 1-3:  SAM (Segment Anything) 体验
Day 4-7:  U-Net 语义分割实战
Day 8-10: YOLOv8 实例分割
Day 11-14: YOLO + DeepSORT 多目标跟踪
```

### 第 7-8 周：深度估计 + 3D
```
Day 1-3:  Depth Anything 单目深度估计
Day 4-7:  双目深度估计 (OpenCV + PyTorch)
Day 8-10: 点云处理入门
Day 11-14: 3D Gaussian Splatting 体验
```

### 第 9-10 周：综合项目
```
选择一个方向做完整项目：
- 自动驾驶感知 (检测+跟踪+深度)
- 机器人视觉 (SLAM + 目标识别)
- 视频分析系统 (检测+跟踪+行为识别)
```
